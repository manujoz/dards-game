# Game Sounds

Please place the following MP3 files in this directory:

- `hit.mp3` (Standard hit)
- `double.mp3` (Double segment hit)
- `triple.mp3` (Triple segment hit)
- `bull.mp3` (Bullseye hit)
- `miss.mp3` (Missed board)
- `win.mp3` (Game win)
- `bust.mp3` (Bust / Over score)
- `change.mp3` (Player change turn)
- `click.mp3` (UI Interaction / Unlock)

These files are loaded by `src/lib/audio/sounds.ts`.
